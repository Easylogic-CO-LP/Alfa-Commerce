# Contributing to Alfa Commerce

Welcome! This guide explains how to contribute to Alfa Commerce, step by step. Whether you're a seasoned developer or just getting started, it covers everything you need to know.

The full, up-to-date contribution guide lives in the **[developer manual](https://manual.alfacommerce.gr/docs/getting-started/contributing)**.

It walks through the prerequisites, the branch workflow, the structural-changes release checklist, a step-by-step GitHub Desktop / Git guide (including the fork path for outside contributors), coding standards, the automated checks, and the pull-request checklist.
